# TyProAction
关于 Ty（专业版）的 Github Action

## 什么是TyPro

你可以在搜索引擎例如 google、baidu、Yandex 等中搜索 TyPro。

也许你找不到 TyPro 是什么，但你可能会找到一些类似的软件、网站或一些其他有用的东西。

:-)

## 如何使用

1. 安装对应版本的TyProAction软件
2. 下载附件的zip文件
3. 解压并移动到 {software-root-dir}\resources
4. 使用keygen模板发起新issue（中英文模板都可以）
5. 享受它
